# Shoppingmall_Lim

국비지원(더조은아카데미) 마지막 과정

* 프로젝트 참여자
  - 임시은
  - 김희관
  - 오상진

* 환경구성
  - jdk 1.8
  - tomcat 7.0
  - oracle 11g
  - Eclipse IDE for Java EE Developers
  
* 구조
  - Dynamic Web Project
  - MVC2 패턴

* 기능
  - 로그인/회원가입/회원수정
  - 상품확인/페이징/장바구니
  - 아임포트 결제기능(데모용, 실제 결제 가능)
  - 관리자) 상품등록/수정/삭제/이미지업로드
